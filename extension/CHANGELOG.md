# Changelog

## 0.1.0 (2026-02-21)

### Features
- **Dictionary Panel** — Search 50K+ real Solodit audit findings with severity filters and pagination
- **Academy Panel** — AI-generated security lessons with interactive quizzes (real code examples from audits)
- **Scout Panel (Ghost Auditor)** — Automated contract analysis with ERC detection, risk pattern cross-referencing, and AI report generation
- **Finding Detail Panel** — Full-width finding reports with PoC code, mitigations, and source links
- **Report Panel** — Severity breakdown, contract analysis card, and cross-referenced findings table
- **Multi-Provider AI** — Supports Gemini (free tier), OpenAI, and Anthropic with auto-detection
- **Backend Auto-Start** — Automatically starts Python backend with health monitoring and status bar indicator
- **Context Menu** — Right-click "Lookup on Solodit" and "Analyze Current File" for .sol files
